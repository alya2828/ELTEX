# Собственный модуль ядра

## Как использовать
1. **Компиляция программы:**
- make

2. **Выполнить втсавку ядра и проверить выходные данные :**
- sudo insmod hello.ko
- sudo dmesg | tail -1
  
3. **Найти свой загруженный модуль ядра:**
- lsmod | grep "hello"

4. **Удалить и проверить модуль ядра:**
- sudo rmmod hello.ko
- sudo dmesg | tail -1
